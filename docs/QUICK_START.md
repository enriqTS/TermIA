# TermIA - Quick Start Guide

## Installation

1. **Clone the repository**:
```bash
git clone https://github.com/enriqTS/TermIA.git
cd TermIA
```

2. **Install dependencies**:
```bash
pip install ply pytest colorama requests pyyaml prompt_toolkit pygments
```

3. **Run TermIA**:
```bash
python main.py
```

## Quick Command Reference

### OS Commands
```bash
pwd                          # Show current directory
ls                          # List files
ls -la                      # List all files with details
cd <directory>              # Change directory
cd ..                       # Go up one directory
cd ~                        # Go to home directory
mkdir <name>                # Create directory
mkdir -p <path>             # Create directory with parents
cat <file>                  # Display file contents
```

### AI Commands
```bash
ia ask "What is Python?"                        # Ask a question
ia summarize "Long text here..."                # Summarize (short)
ia summarize "Text..." --length medium          # Summarize (medium)
ia codeexplain main.py                          # Explain code file
ia translate "Hello World" --to pt              # Translate to Portuguese
```

### Control Commands
```bash
history                     # Show last 10 commands
history 20                  # Show last 20 commands
clear                       # Clear screen
help                        # Show all commands
help ls                     # Show help for specific command
exit                        # Exit TermIA
```

## Enhanced Features

### Autocomplete (Tab)
- Press `Tab` to autocomplete commands, options, and paths
- Works for all commands and subcommands
- Smart suggestions based on context

### Syntax Highlighting
- Commands are color-coded as you type
- Different colors for commands, options, strings, paths

### Keyboard Shortcuts
- `Ctrl+A`: Jump to beginning of line
- `Ctrl+E`: Jump to end of line
- `Ctrl+K`: Delete to end of line
- `Ctrl+L`: Clear screen
- `Ctrl+C`: Cancel input
- `Ctrl+D`: Exit

### AI Commands
```bash
TermIA> ia ask "What is a compiler?"
[IA] Processing question...

TermIA> ia summarize "Python is a high-level programming language..." --length short
[IA] Summarizing text (size: short)...

TermIA> ia translate "Good morning" --to pt
```

### Using Autocomplete
```bash
TermIA> l<Tab>              # Suggests: ls
TermIA> ls -<Tab>           # Suggests: -a, -l, -h, -la, -lh, -lah
TermIA> ia <Tab>            # Suggests: ask, summarize, codeexplain, translate
TermIA> cat READ<Tab>       # Suggests: README.md
```

## Troubleshooting

### "Module not found" error
Install missing dependencies:
```bash
pip install ply colorama requests pyyaml prompt_toolkit pygments
```

### Syntax highlighting not working
Make sure Pygments is installed:
```bash
pip install pygments
```

### Enhanced features disabled
This is normal on some terminals. The terminal will fallback to basic mode automatically.
To force basic mode:
```bash
python main.py --basic  # (if implemented)
```

### Command not recognized
- Type `help` to see all available commands
- Check command syntax with `help <command>`
- Use Tab completion to discover available options

## Tips & Tricks

1. **Use Tab liberally**: Press Tab after any command or partial word to see suggestions

2. **Search history**: Press `Ctrl+R` and start typing to find previous commands

3. **Quick navigation**:
   - `cd ..` to go up
   - `cd ~` to go home
   - `pwd` to see where you are

4. **AI commands**: Enclose text in quotes for AI commands:
   ```bash
   ia ask "Your question here"
   ```

5. **View history**: Use `history 50` to see your last 50 commands

6. **File operations**:
   - Use `ls -la` for detailed file listings
   - Use `cat` to quickly view file contents
   - Use Tab to autocomplete file paths

## Debug Mode

Run with debug mode to see tokens and AST:
```bash
python main.py --debug
```

This will show:
- Tokens generated by the lexer
- Abstract Syntax Tree (AST) structure
- Detailed error information
